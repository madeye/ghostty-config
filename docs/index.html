ghostty-config

A web-based configuration GUI for the Ghostty terminal emulator. Browse options, preview themes, and manage keybindings — all from your browser.

GitHub Quick Start Download Releases

Features

Everything you need to configure Ghostty without hand-editing config files.

Config Editor

Browse and edit all 180+ Ghostty config options organized by category.

400+ Themes

Preview and apply bundled themes with a live visual preview.

Keybindings

Manage keybindings with a visual key capture UI.

Terminal Preview

See your configuration changes rendered in a live terminal preview.

Validation

Config validation catches errors before they reach Ghostty.

Save & Apply

Import/export config and auto-reload Ghostty on save (macOS).

Quick Start

Up and running in under a minute.

Requirements

  • Ghostty installed
  • Rust toolchain

Run

$ cargo run --releaseOpens automatically at http://127.0.0.1:3456
